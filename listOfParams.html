Parameter Description Valid values
tag Your Associate id e.g. widgetsamazon-20
width Widget width in pixels Valid combinations of Width x Height are: 120x500, 160x600, 400x150, 500x175, 600x200
height Widget height in pixels
title Widget title String value up to 60 characters long
list Displays bestsellers, new releases etc. from Amazon Valid values include:
Bestsellers
NewReleases
MostWishedFor
MostGifted
keywords Search phrase used to search for products on Amazon and populate your Carousel. Any string value.
category Determines the Amazon category used for adding products to the widget. Only applicable when 'list' or 'keywords' is specified.
What is a Search Index?		 Any valid Amazon SearchIndex. Click here for a list of valid SearchIndex values
browseNode Determines the Amazon sub-category used for adding products to the widget. Only applicable when 'list' or 'keywords' is specified.
What is a Browse Node?		 Any valid Amazon BrowseNodeId. Click here for a list of valid BrowseNode values
asins Comma-separated list of ASINs that will be added to the widget. All valid Amazon ASINs
shuffleProducts Set to True if you want the widget to show products in a random order each time it is displayed True, False
showBorder Set to True if you want a black border around the widget True, False
